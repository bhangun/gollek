# GGUF Adapter (llama.cpp)

This adapter provides native inference support for GGUF/ggml models via llama.cpp bindings.

## Key Capabilities

* GGUF artifact loading
* CPU/CUDA support
* Token generation via llama.cpp

## Key Paths

* Binding: `inference-golek/adapter/golek-inference-gguf/src/main/java/tech/kayys/golek/inference/gguf/LlamaCppBinding.java`
* Runner: `inference-golek/adapter/golek-inference-gguf/src/main/java/tech/kayys/golek/inference/gguf/LlamaCppRunner.java`
* Config: `inference-golek/adapter/golek-inference-gguf/src/main/java/tech/kayys/golek/inference/gguf/GGUFConfig.java`
