Gollek Unified Runtime

Model Inference

Available Endpoints

  • Models API - List available models
  • Inference API - Run model inference
  • OpenAPI Specification - API documentation
  • Swagger UI - Interactive API documentation
  • Health Check - Service health status
  • Metrics - Service metrics

CLI Access

You can also access the CLI functionality by running the JAR with command-line arguments:

java -jar gollek-runtime-unified-1.0.0-SNAPSHOT-runner.jar [command]

Example commands:

  • java -jar gollek-runtime-unified-1.0.0-SNAPSHOT-runner.jar models list - List available models
  • java -jar gollek-runtime-unified-1.0.0-SNAPSHOT-runner.jar inference --model <model> --prompt <prompt> - Run inference
